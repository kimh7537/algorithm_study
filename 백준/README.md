# 전역변수를 함부로 사용하지 말기
- 파트3 (97)컴백홈의 ny, nx가 전역변수라면 값이 틀림
- `visited[ny][nx] = visited[y][x] + 1; ret += go(ny, nx); visited[0][0] = 0;`과 같은 코드에서 재귀함수가 끝날때(visited[0][0] = 0) 문제가 발생함
- **함수의 매개변수**는 지역변수를 많이 사용할 것 고려
